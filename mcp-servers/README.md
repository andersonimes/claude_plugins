# MCP Servers

Model Context Protocol (MCP) servers provide additional tools and resources to Claude Code.

## What are MCP Servers?

MCP servers extend Claude Code's capabilities by providing:
- External API integrations
- Database connections
- File system access
- Custom tools and functions

## Available Servers

Currently empty - be the first to contribute an MCP server!

## Contributing

To add your MCP server:

1. Create a directory with your server name
2. Include all server code and dependencies
3. Add a comprehensive README with:
   - Description
   - Installation instructions
   - Configuration steps
   - Usage examples
4. Submit a pull request

See [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.

## Installing MCP Servers

To install an MCP server from this marketplace:

1. Navigate to the server's directory
2. Follow the installation instructions in its README
3. Configure the server in your Claude Code settings
4. Restart Claude Code
