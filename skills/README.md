# Skills

Skills are reusable capabilities that can be invoked within Claude Code conversations.

## What are Skills?

Skills provide specialized functionality such as:
- Code analysis and refactoring
- Documentation generation
- Testing utilities
- Domain-specific knowledge

## Available Skills

Currently empty - be the first to contribute a skill!

## Contributing

To add your skill:

1. Create a directory with your skill name
2. Include the skill definition and any required files
3. Add a README with:
   - Description
   - Installation instructions
   - Usage examples
   - Configuration options
4. Submit a pull request

See [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.

## Installing Skills

To install a skill from this marketplace:

1. Navigate to the skill's directory
2. Follow the installation instructions in its README
3. Skills are typically copied to your Claude Code skills directory
4. The skill will be available for use in your conversations
