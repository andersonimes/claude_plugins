# Claude Code Plugin Marketplace

A community-driven marketplace for Claude Code plugins, MCP servers, skills, and slash commands.

## What's Inside

This marketplace contains various plugins to extend Claude Code functionality:

- **MCP Servers** - Model Context Protocol servers that provide additional tools and resources
- **Skills** - Reusable skills that can be invoked within Claude Code
- **Slash Commands** - Custom commands for quick actions

## How to Use

Browse the directories above to find plugins that interest you. Each plugin includes:
- Installation instructions
- Usage examples
- Configuration details

## Plugin Categories

### MCP Servers (`/mcp-servers`)
MCP servers provide external integrations and tools for Claude Code.

### Skills (`/skills`)
Skills are specialized capabilities that can be invoked during conversations.

### Slash Commands (`/slash-commands`)
Quick commands for common tasks and workflows.

## Contributing

We welcome contributions! To add your plugin:

1. Fork this repository
2. Add your plugin to the appropriate directory
3. Include a README with installation and usage instructions
4. Submit a pull request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.

## Getting Started

If you're new to Claude Code plugins, start with the example plugins in each directory to understand the structure.
